- [ ] The ingestion `sourceCommit` is the REAL `createCommitActivity` (over `applyPlan`), NOT the in-memory fake — a validated ingestion plan commits to canonical Markdown and returns a real `revisionId`.
- [ ] The source `KnowledgeWriterDeps` bind `{ vault: backends.vault, revisions: <the 2a durable store>, audit: backends.repos.audit }` with `ownershipCheck` + `secretScan` **LEFT UNSET** → the REAL defaults (`enforceHumanOwnership` + `scanForSecrets`) — NEVER a pass-through (the writer's sole-writer + secret guards are live).
- [ ] `expectedBaseRevision` is a **resolver** (`() => readVaultHeadRevision(vault)`) that reads the LIVE vault head (the vault moves), not a fixed value; `deriveIdempotencyKey(plan) = "kw:commit:" + String(plan.planId)` (matches the meeting pattern).

7. **`source_commit_plan_targets_bound_workspace`** — Asserts: the committed revision's `AuditRecord.workspaceId` == the routing-bound workspace; a plan whose workspaceId ≠ bound is rejected. Why: WS-8 (derive-from-validated).
