**14.2 connectors surface:**
- [ ] Lists the current workspace's connector instances (empty-until-data) and offers register / enable / pause / set-cadence, calling `connectorConfig`.

- [ ] Surfaces routed/mounted in AppShell; deterministic-flow tests pass (component/command-caller tests); `/preflight` clean.
