## Pinned decisions (load-bearing outputs)

- **0.1 Electron:** `electron` 42.5.1, `electron-builder` 26.15.3; V1 target = macOS `dir`, unsigned (`mac.identity=null` + `CSC_IDENTITY_AUTO_DISCOVERY=false`); child spawn via `process.execPath` + `ELECTRON_RUN_AS_NODE=1` or bundled binaries, one Supervisor per workspace for GBrain; supervision defaults backoff min200/max30000/×2, crash-loop 5/60s → terminal "down", stableUptime 10s resets. Hardened-runtime entitlements list captured for V1.1 signing (`disable-library-validation` load-bearing).
- **0.2 GBrain:** engine = `garrytan/gbrain` (PGLite); V1 posture **read-only/index-only**; `config/gbrain.pin` format `key = value`; SHA `PENDING_LIVE_VALIDATION` (fail-closed); candidate `08b3698e90532b7b66c445e6b1d8cdfe71822802` (v0.18.2, gstack known-good); pin-bump gate = §12 parity suite green + full re-index (§13).
- **0.3 Hermes:** v0.17.0; primary surface = one-shot CLI subprocess `hermes chat -q <prompt> -Q -t <readonly> -m <model> --provider <p> --max-turns N`; RT-7 autonomous = Kanban; structured fallback = ACP. ⚠️ **read-only runs MUST pass an explicit minimal toolset — empty `-t` silently falls back to the user's full (mutating) config toolset.**
- **0.4 Providers:** default models — `meeting.close`=`claude-opus-4-8` (claude-agent-sdk runtime), `extraction`/`synthesis`/`copilot.qa`=`claude-sonnet-4-6`, `cheap.classify`=`claude-haiku-4-5`; local zero-egress fallbacks via Ollama (deferred). `maxCostUsd` global 0.50. **Live-verified passing (via OpenRouter):** `deepseek/deepseek-v4-pro` ($0.435/$0.87 per Mtok), `anthropic/claude-haiku-4.5` ($1/$5). OpenRouter recorded as its **own processor**, never an OpenAI alias.
- **0.5 Streaming:** tRPC v11 subscription over **WebSocket** (`applyWSSHandler`/`wsLink`); commands/queries over `httpBatchLink`; resumability via `tracked(eventId)` + `lastEventId` + bounded replay buffer; consumers **idempotent by event id (at-least-once)**; per-launch session token via `connectionParams` validated in `createContext` pre-handler. SSE (`httpSubscriptionLink`) is the validated drop-in fallback (router is transport-agnostic).
- **0.6 Perf:** hard gates — dashboard <2s, KW→GBrain ≤60s p95, KW→dashboard ≤10s p95. SLOs — meeting-closeout e2e p95 ≤180s cloud/600s local; Copilot TTFT ≤3s, full ≤20s/45s; approval state-reflect ≤2s, external-write ≤10s; ingress→wf-start ≤2s. Concurrency 4 cloud / 1 local AgentJobs; Temporal 16 activity / 32 wf-task slots. `maxCostUsd` per-capability rows; local runtime ×3 multiplier.
